Neb AI

Neb AI is a desktop assistant built with Python, Tkinter, SQLite, and bcrypt. It provides a simple local authentication system with signup and login, then opens a basic assistant interface that responds to wake-word style input.
This project is a good starting point for building a personal desktop assistant with:
local user accounts
password hashing
a graphical interface
session-based identity awareness
room for future LLM integration
Features
User signup and login
Secure password storage with bcrypt
Local SQLite database
Personalized greeting using display name
Wake word activation with "Hey Neb"
Basic assistant commands:
ask its name
ask your name
put it back to sleep
Simple structure for extending assistant logic
